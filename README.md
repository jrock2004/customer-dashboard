This is a POC to show off some of the abilities that you can do with NextJS.

## Requirements

- NodeJS
- Postgres Server
- pnpm

## Getting Started

1. Clone the repository
2. Run `pnpm install` to install the dependencies
3. Update the .env file to put in values needed to connect to your Postgres server
4. Run `pnpm dev` to start the development server

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
